## How to install 

* Install neovim:
```bash
sudo snap install nvim --classic
``` 
* clone this repo to config folder:
```bash
git clone https://github.com/se7oluti0n/nvim_setup ~/.config/nvim
```
* Install dependencies
```bash
cd ~/.config/nvim
sh install_dependencies.sh
```
* enjoy nvim
```bash
cd [your_workspace]
nvim
```
